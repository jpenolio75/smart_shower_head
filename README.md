# Smart Shower Head

A cross-platform companion mobile app for Smart Shower Head, a UNLV Senior Design Project (EE 497-498).

## Features
- Display water temperature and flow rate on the LCD screen and the app.
- Allow user to set a specified water temperature, shower time limit, and water usage; app will notify them once it is reached.
- Play music on Spotify.
- Make emergency calls in case of injury (e.g., slipping and falling in the shower).
- User will be able to tell the app to perform all of those functions through voice using Alan AI.
